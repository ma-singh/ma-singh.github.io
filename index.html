Projects

  • Notifications

    The goal of this project was to create a plugin that could be used to display notifications for various environments. It was made using jQuery, and it includes basic functions to create a blank <div> to overlay the page, style the notifications and then implement the messages themselves, and finally animate and display the whole component.

  • Weather App

    This is a pseudo-weather app that I created as a final project for a practical exam. It utilizes the Weather Underground API to asynchronously load and display information about the current weather conditions in my hometown and an additional location.

  • The Safari Zone

    The Safari Zone represents my first foray into the world of HTML5 structure, CSS styling, and responsive design. We were given free reign on the theme so long as we were able to create something with structure (navigation, headings, images, text, forms) styling it appropriately, and filling it with sufficient content that we could then have the page react responsively.

  • HappyHour

    HappyHour was the result of a project where we were tasked to create mockups for an application you could see having potential use. I chose to design a social drinking application that would allow you to find either specific alcoholic beverages or popular locations serving alcohol nearby, meet up with friends, and review said beverages or locations.

  • World of Warcraft Tool

    For this project we were tasked with utilizing an existing API to accomplish a task. I took Blizzard's API for World of Warcraft to pull character and auction house data from a given realm and display what the quantity, average, and prices for a selection of items were being sold for. This was created with HTML5, JavaScript, jQuery, and CSS3.

  • interNASHional E-Commerce

    interNASHional E-Commerce is an application designed to simulate a shopping cart type application that allows users to purchase comic books, with an Admin CMS to manage inventory. It uses NodeJS, Express. The Stripe API takes care of simulating payments, while PassportJS handles user authentication. MongoDB and Mongoose are used to manage the database.
